Política de Privacidad

Esta aplicación utiliza la API de WhatsApp Business para recibir y responder mensajes mediante automatización con n8n.

Información recopilada

  • Nombre del usuario.
  • Número de WhatsApp.
  • Mensajes enviados.

Uso de la información

La información se utiliza únicamente para responder mensajes y prestar el servicio.

Compartición de datos

No vendemos ni compartimos información personal con terceros, salvo cuando sea necesario para el funcionamiento de los servicios de Meta y OpenAI.
